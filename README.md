# Minecraft Bedrock Addon Workspace

Dette workspace er sat op til Minecraft Bedrock 1.21.132 og indeholder:

- `behavior_packs/` — til dine behavior packs (logik, scripts, entities, osv.)
- `resource_packs/` — til dine resource packs (textures, lyd, UI, osv.)

## Kom i gang
1. Læg dine addon-filer i de relevante mapper.
2. Rediger `manifest.json` i hver mappe for at tilpasse navn, beskrivelse og UUIDs.
3. Kopiér mapperne til din Minecraft `com.mojang`-mappe for at teste dem i spillet.

## Eksempel
Der er eksempler på `manifest.json` i begge mapper.

## Struktur
```
behavior_packs/
  manifest.json
resource_packs/
  manifest.json
.github/
  copilot-instructions.md
README.md
```
